Add an easy-to-use spam filter to your comment form.

== Installation ==

= Installing: =

1. Download the package.
2. Extract it to the "plugins" folder of your WordPress directory.
3. In the wp-admin panel, go to "Plugins" and activate it.
4. The filter will automatically be placed in each of your comment forms. 
   Spammy comments will show in your spam folder under "Comments" in the wp-admin.
   
= Uninstalling: =

1. In the wp-admin panel, go to "Plugins" and deactivate the plugin.
2. Go to the "plugins" folder of your WordPress directory and delete the SimpleSpamFilter folder.
